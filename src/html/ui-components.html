Window scrolling

Turn any anchor tag into a window scrolling trigger.
Just pass in an ID or CLASS selector as a data attribute, and you're good to go! 
Attributes:  data-ws-el, data-ws-duration
Class hook:  .ws-trigger

Video overlay

Turn any anchor tag into a video overlay trigger.
You can use these triggers to play a Vimeo video {{ sprite('pointer-right--yellow', classes='icon') }} or anything from Youtube. 
Attributes:  data-vo-type, data-vo-id
Class hook:  .vo-trigger

Video embed

Video embeds are fully responsive and always take up 100% of their parent's width. The videos are fitted to 16:9 ratio, but it can be changed from CSS.
